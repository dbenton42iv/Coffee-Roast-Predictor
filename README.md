# Coffee Roast Predictor

This project aims to predict the perfect roast profile for coffee based on various input parameters.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/dbenton42iv/Coffee-Roast-Predictor.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Coffee-Roast-Predictor
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To run the predictor, execute:
```bash
python predictor.py
```
